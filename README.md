## ğŸ› ï¸ How to Get Started

Ready to jump in? Follow these simple steps to get the app running on your local machine.

### **Prerequisites**

Make sure you have [Node.js](https://nodejs.org/) and [pnpm](https://pnpm.io/) installed.

### **Installation & Setup**

1.  **Clone the repo:**

    ```bash
    git clone https://github.com/ashu-dwd/learning-ci-cd.git
    cd learning-ci-cd
    ```

2.  **Install dependencies:**

    ```bash
    pnpm install
    ```

3.  **Start the server:**

    ```bash
    pnpm start
    ```

4.  **Open your browser** and navigate to `http://localhost:3000`. You should see the app live! ğŸ¥³
